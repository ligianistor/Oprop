JExpr - a parser for a very small subset of Java. It was created to help me understand what would be necessary to develop a full compiler starting from a JavaCC-generated parser - in effect to answer the FAQ "TODO".

Only a very small subset of Java can be compiled - see expr.txt as an example.

This system consists of the following components:

  • A JavaCC parser, JExpr.jj - see {@linkplain cager.jexpr.JExpr}
  • A set of {@linkplain cager.jexpr.ast Abstract Syntax Tree} classes.
  • A {@linkplain cager.jexpr.visitor.ContextVisitor Contextual (semantic) Analsys visitor}.
  • An {@linkplain cager.jexpr.Operator} type resolution class used by the Contextual Visitor.
